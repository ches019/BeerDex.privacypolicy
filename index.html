Política de Privacidad de Beer.db

Última actualización: 11 de marzo de 2026

Esta página informa a los usuarios sobre nuestras políticas de recopilación, uso y divulgación de información personal si alguien decide utilizar nuestra aplicación.

1. Recopilación de Información

Nuestra aplicación utiliza la base de datos BJCP para fines informativos y educativos. Si utilizas funciones de sincronización, podríamos utilizar servicios como Google Firebase o Supabase que podrían recopilar identificadores de dispositivo de forma anónima para el funcionamiento técnico.

2. Datos de la Guía BJCP

La información sobre estilos de cerveza, hidromiel y sidra es propiedad intelectual del Beer Judge Certification Program (BJCP). Esta aplicación se ofrece como una herramienta de consulta gratuita.

3. Seguridad

Valoramos tu confianza al proporcionarnos tu información, por lo que nos esforzamos por utilizar medios comercialmente aceptables para protegerla.

4. Cambios a esta Política

Podemos actualizar nuestra Política de Privacidad periódicamente. Se recomienda revisar esta página para ver si hay cambios.
